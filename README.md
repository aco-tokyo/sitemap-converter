# Sitemap Converter

Integrity Plusから出力したCSVファイルを、

| URL                      | Title | 階層1                | 階層2 | 階層3 |
| ------------------------ | ----- | -------------------- | ----- | ----- |
| https://example.com/     | Root  | https://example.com/ |       |       |
| https://example.com/a/   | A     | https://example.com/ | a/    |       |
| https://example.com/a/b/ | B     | https://example.com/ | a/    | b/    |

の形式に変換するコンバーターです。

スプレッドシートアプリで開くとこうなります：
